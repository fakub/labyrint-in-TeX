
# Generátor Labyrintu

Skript původně od někoho z Fykosu, vlastní úpravy.


## Použití

Jak postupovat:

 1. uprav soubory `*.tex` dle libosti,
 2. zkompiluj (potřebuješ plain TeX):

```$ pdfcsplain labyrint.tex```

Hotovo!
